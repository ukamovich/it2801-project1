The person behind this page and two art figures has been inspired by simple forms. The SVG art object is the result of visiting the exhbition of Kazimir Malevich (a Russian avant-garde artist) in the Tate Gallery in London back in 2016. The Canvas art object is an attempt to create a new emoji face – What Face – since the author is a big fan of emojis. In addition, he got inspired by existing code snippets to create three squares, where one of them is toggling up and down, and can change color using a button.

Currently, the author of this page studies Computer Science at NTNU and is looking forward to work on a project with other group mates. IT2810 looks like a very exciting subject with a lot of hands-on learning, which is awesome.

Question 1: I decided to focuss on using CSS-grid only solution across this page. My choice is determined by the desire to master one grid layout at the moment. First, I used a piece of paper to roughly design how the page will look like. Second, I divided the page into a grid with sections using columns and rows. Then, I created class "container" and positioned each section using 'grid-template-areas'. Each area has a unique id (#about, #svg, #canvas, #docs) and its styling.

Question 2: For SVG figure, I did 'hard coding' to draw vector-based figure in a separate file with extention .svg. After the figure was ready, I simply embedded the SVG code snippet inside the #svg area. To animate the SVG figure, I used @keyframes in style.css, while for interactivity (changing colors with buttons), I used jQuery and wrote a script in scripts.js file. For HTML canvas figure, I used the canvas element to embed inside the #canvas area. For actual drawying, I used Canvas API, which provides the posibility for drawing 2D graphics via JavaScript. Since I had difficulaties to implement animation and interactivity for the HTML canvas figure (What Face), I creted a separate figure (3 squares), and wrote a script to show animation and interactivity.

Question 3: Since I did not know much about jQuery before this assignmet, I read and watched a few tuturials about the differences between jQeury and JavaScript. It was clear that jQuery is much easier to use for manipulation and animation of both SVG and HTML canvas art figures. I used jQuery for writing scripts to manipulate the figures and to reveal/hide answers to the required questions.

Question 4: I used 3 browsers (Firefox, Safari, Chrome) to test the page. Unfortunately, the content of the page is not displayed well under Safari and Chrome while changing the size of the brower window or zooming in and out. Ideally, Firefox should be usedd to avoid visual inconvenience.
